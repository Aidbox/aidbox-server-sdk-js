## Requirements

- NodeJS 16 or above
- Aidbox Instance (Local or Cloud)


## Install

```npm
pnpm install @aidbox/node-server-sdk
```
or 
```npm
npm install @aidbox/node-server-sdk
```
or
```npm
yarn add @aidbox/node-server-sdk
```
